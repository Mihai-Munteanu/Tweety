# PROJHECT_NAME

This project's name is: Tweety

## Install

```
git clone https://github.com/Mihai-Munteanu/Tweety
composer install
npm install
```

Create a DB named `tweety`.

You must run seeder with `php artisan db:seed`. This will create a user having the following credentials: user:Admin, username: Admin, email: admin@example.com password:123456789

Then by accessing the following links you may use the app as follows:

-   http://tweety.test - welcome page;
-   /register - register page;
-   /login - login page;
-   /tweets - home page. You can see all your and your followee tweets.
-   /explore - see the entire list of users from where can be selected users to follow.
-   /profiles/{user:username} - see a user profile and start following him/her;
-   /profiles/{user:username}/edit - edit your own profile;
